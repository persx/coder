# Manual Label Creation (No gh CLI needed)

If you don't have `gh` CLI installed, create labels manually:

## Create Each Label via Web UI

Visit: https://github.com/persx/persx/labels/new

Create these 4 labels one by one:

---

### Label 1: bot
- **Name:** `bot`
- **Description:** `Automated changes from Claude bot`
- **Color:** `#0e8a16` (green)
- Click **"Create label"**

---

### Label 2: safe-change
- **Name:** `safe-change`
- **Description:** `Low-risk change verified by guardrails`
- **Color:** `#bfdadc` (light blue)
- Click **"Create label"**

---

### Label 3: needs-human
- **Name:** `needs-human`
- **Description:** `Requires human review and decision`
- **Color:** `#fbca04` (yellow)
- Click **"Create label"**

---

### Label 4: blocked
- **Name:** `blocked`
- **Description:** `Bot cannot proceed - manual intervention needed`
- **Color:** `#d93f0b` (red)
- Click **"Create label"**

---

## Verify Labels Created

Visit: https://github.com/persx/persx/labels

You should see all 4 new labels listed.
